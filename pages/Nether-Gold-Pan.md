# 下界淘金盘 :id=nether-gold-pan

下界淘金盘是[工具](/Tools)分类中的物品。

## 如何使用下界淘金盘 :id=how-to-use-the-nether-gold-pan

当手持下界淘金盘时，右键点击灵魂沙方块。此时灵魂沙方块会消失，然后会掉落石英、金粒、下界疣、烈焰粉、萤石粉或恶魂泪中的一种。

## 掉落几率 :id=drop-chances

通过下界淘金盘获得物品的几率如下表所示：

| 物品 | 几率 |
| ---- | ------ |
| 石英 | 50% |
| 金粒 | 25% |
| 下界疣 | 10% |
| 烈焰粉 | 8% |
| 萤石粉 | 5% |
| 恶魂泪 | 2% |
